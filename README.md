## 📖 Utilisation

### Exemple Basique

```python
from nanoqr import NanoQREncoder, NanoQRDecoder, NanoQRVisualizer

# Créer l'encodeur et décodeur
encoder = NanoQREncoder(grid_size=7)
decoder = NanoQRDecoder(grid_size=7)

# Encoder un message
message = "QUANTUM_SECURE_2024"
nanoqr = encoder.encode(message)

# Visualiser à différents niveaux de zoom
print(NanoQRVisualizer.visualize(nanoqr, zoom_level=1))   # Point simple
print(NanoQRVisualizer.visualize(nanoqr, zoom_level=100)) # Complexité totale

# Décoder le message
decoded = decoder.decode(nanoqr)
print(f"Decoded: {decoded}")
```

### Encodage de Clé Cryptographique

```python
# Encoder une clé cryptographique
crypto_key = "9F7E2A8B1C4D6E3F5A0B"
nanoqr = encoder.encode(crypto_key, use_error_correction=True)

# Analyse de sécurité
from nanoqr import SecurityAnalyzer
SecurityAnalyzer.print_security_report(nanoqr)
```
